> 조건문(if ~ else)
>> 두 개 이상의 인자값이 서로 타입이 다를 경우
```
def plus(a, b):
  if type(b) is str:
    return None
  else:
    return a + b

plus(12, "10") # None(아무것도 안뜸)
print(plus(12, 10)) # 22
```
>> 정수나 실수가 들어올 경우에 연산하는 경우
```
def plus(a, b):
  if type(b) is int or type(b) is float:
    return a + b
  else:
    return None

print(plus(12, 1.2)) # 13.2
print(plus(12, 1)) # 13
plus(12, "awos")) # None(아무것도 안뜸)
```
