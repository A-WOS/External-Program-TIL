> 함수 사용법
>   > 들여쓰기로 함수 판단
>   >   > 파라미터 값이 없는 경우    
    
    def say_hello():
        print("hello")
    
    say_hello() # hello 
    
>   >   > 파라미터 값이 있는 경우
```
def say_hello(who):
    print("hello", who)

say_hello("Awos") # hello Awos
```
```
def plus(a, b):
    print(a + b)
        
plus(2, 5) # 7
```
```
def minus(a, b):
    print(a - b)

minus(2, 5) # -3
```
>  >  > 파라미터 값이 있고 default 값이 있는 경우
```
def say_hello(name="anonymous"):
    print("hello", name)

say_hello() # hello anonymous
say_hello("awos") # hello awos
```
